Fitur Aplikasi:


✅ SMS Om 

  Deskripsi: Melihat SMS lama yang tersimpan di perangkat korban.


✅ Kontak 

  Deskripsi: Melihat daftar kontak yang di save korban.


✅ Call Log 

  Deskripsi: Melihat riwayat panggilan telepon yang dilakukan oleh korban.


✅ Gallery (Bug) 

  Deskripsi: Melihat foto dan video yang tersimpan di galeri perangkat korban.


✅ Sistem 

  Deskripsi: Memantau status sistem, seperti penggunaan CPU, memori, dan baterai.


✅ File 

  Deskripsi: Melihat file yang tersimpan di perangkat korban.

  Deskripsi: Menampilkan teks pada layar perangkat korban.

✅ Hapus file 

  Deskripsi: Menghapus semua file di perangkat korban.


✅ Senter 

  Deskripsi: Menghidupkan senter pada perangkat korban.


✅ Play Musik 

  Deskripsi: Memutar musik dari perangkat korban.


✅ Getaran 

   Deskripsi: Mengaktifkan getaran pada perangkat korban.


✅ Ubah Wallpaper 

   Deskripsi: Mengubah wallpaper layar utama & kunci perangkat korban.


✅ Notifikasi 

   Deskripsi: Mengirim notifikasi pada perangkat korban.
